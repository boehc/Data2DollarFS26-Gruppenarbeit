📋 SCHRITT 1: API KEY EINRICHTEN
────────────────────────────────────────────────────────────────────────────────

1. Gehe zu: https://platform.openai.com/account/api-keys
2. Erstelle einen neuen API Key (oder kopiere einen bestehenden)
3. Öffne PowerShell und führe aus:
   
   $env:OPENAI_API_KEY = "sk-your-api-key-here"
   
   (Ersetze "sk-your-api-key-here" mit deinem echten Key!)

✅ TIP: Den Key in der PowerShell mit: echo $env:OPENAI_API_KEY überprüfen


📦 SCHRITT 2: ABHÄNGIGKEITEN INSTALLIEREN
────────────────────────────────────────────────────────────────────────────────

Öffne PowerShell im Verzeichnis und führe aus:

   pip install openai pandas tqdm --upgrade


🔍 SCHRITT 3: SETUP VALIDIEREN (OPTIONAL)
────────────────────────────────────────────────────────────────────────────────

Bevor du das Hauptskript ausführst, überprüfe die Einrichtung:

   python validate_setup.py

Ergebnis sollte sein: "🎉 ALL CHECKS PASSED!"


▶️  SCHRITT 4: HAUPTSKRIPT AUSFÜHREN
────────────────────────────────────────────────────────────────────────────────

   cd "c:\Users\nadja\1_Data2Dollar\Kurskatalog"
   python extract_skills_with_openai.py
